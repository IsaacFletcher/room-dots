# My Hyprland Dotfiles

My Arch Linux + Hyprland + Swaync + Waybar Dotfiles 

## Dependencies

```
waybar
swaync
rofi
qt6ct
qt5ct
dolphin
gtk-3.0
gtk-4.0
kvantum
cliphist
```

## Installation Guide

Arch Linux:

```
git clone https://github.com/isaacfletcher/room-dots.git
chmod +x install.sh
./install.sh
```
